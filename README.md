# MetroNear

MetroNear is a web application that helps users find the best metro station to reach their destination in Bangalore. The application automatically recommends the most convenient metro station based on real road travel time and distance.

## Project Structure

```
MetroNear/
├── index.html              # Main HTML file with wireframe layout
├── README.md               # Project documentation
├── src/
│   ├── css/
│   │   └── style.css       # Stylesheet for the application
│   ├── js/
│   │   └── main.js         # Main JavaScript file
│   └── assets/             # Images and other assets
```

## Features

- Search for destinations in Bangalore
- Visual map display with metro stations
- Recommendations based on travel time
- Responsive design for mobile and desktop
- Clean, intuitive user interface

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open `index.html` in your browser to run the application locally

## Development

This project uses:
- Vanilla JavaScript (no frameworks)
- Leaflet.js for maps
- OpenStreetMap for map tiles
- Responsive CSS
